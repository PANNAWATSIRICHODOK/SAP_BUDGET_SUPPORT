# SAP_BUDGET_SUPPORT

โปรเจกต์นี้ใช้สำหรับนำเสนอ 2 มุมมองของ Budget Add-on ใน SAP Business One

- `pages/addons-budget-flow.html` : `flow ของ addons budget`
- `pages/problem-report.html` : `รายงานปัญหา`
- `index.html` : entry page สำหรับ GitHub Pages และ redirect ไปหน้า flow

ไฟล์หลักของเว็บ

- `pages/addons-budget-flow.html`
- `pages/problem-report.html`
- `assets/css/addons-budget-flow.css`
- `assets/css/problem-report.css`
- `assets/js/addons-budget-flow.js`
- `assets/js/problem-report.js`
- `index.html`

โฟลเดอร์อ้างอิง SQL

- `references/NDBS_UAT_JBF`
- `references/NDBS_SBO_JBF`

หน้าแรกของเว็บจะพาไป `flow ของ addons budget` ก่อน และสามารถสลับไป `รายงานปัญหา` ได้จากปุ่มด้านบน
